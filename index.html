このガイドの要点

メタマスクは、イーサリアムをはじめとする主要ブロックチェーンに対応した、世界で最も広く使われているセルフカストディ型の暗号資産ウォレットの一つです。利用を開始するには、まず公式サイト metamask.io にアクセスし、利用環境に合わせて Chrome・Firefox・Edge・Brave 向けのブラウザ拡張機能、または iOS・Android 向けのモバイルアプリをダウンロードします。インストール後はアプリを起動し、「新しいウォレットを作成」を選んでパスワードを設定し、12 語のシークレットリカバリーフレーズを紙に書き留めてオフラインで厳重に保管します。続く確認テストに合格するとウォレットが有効化され、初回ログインは設定したパスワードで行います。本ガイドでは、偽アプリの見分け方、各プラットフォーム別のインストール手順、初期設定の細部、ログインの流れ、資産を守るために欠かせないセキュリティ上の注意点、よくあるトラブルの対処までを一通りまとめています。初めての方でも、この一本で安全にメタマスクを使い始められるように構成しています。

メタマスクとは何か?

メタマスク(MetaMask)は、米国 ConsenSys Software 社の関連組織が開発・運営する、イーサリアム仮想マシン(EVM)互換チェーンに対応した暗号資産ウォレットです。2016 年に初版が公開されて以来、Web3 分野の標準的なウォレットとして定着し、2024 年時点で月間アクティブユーザー数は全世界で 3,000 万人規模に達しているとされます。ブラウザ拡張機能と iOS・Android 向けモバイルアプリの二系統が提供されており、端末側に秘密鍵を保存するセルフカストディ(非託管)型である点が最大の特徴です。

取引所のウォレットとは異なり、資産の所有権と移動の最終判断は常に利用者自身にあります。この性質は自由度と引き換えに大きな責任を伴うため、本ガイドはセキュリティ面の注意点にも十分な紙幅を割いています。

対応ネットワークはイーサリアム本体だけにとどまらず、Polygon、BNB Smart Chain、Arbitrum、Optimism、Base、Avalanche といった主要 L2・EVM 系チェーンを手動または自動で追加して切り替え可能です。NFT マーケットプレイス、DeFi プロトコル、Web3 ゲーム、DAO プラットフォームなど、ほぼすべての dApp が標準的なログイン手段としてメタマスクを受け入れています。

メタマスクはどこから安全にダウンロードできるか?

結論から言うと、メタマスクのダウンロード元は一箇所しかありません。公式サイト metamask.io、およびそこから案内されるブラウザの公式ストア(Chrome ウェブストア、Firefox Add-ons、Microsoft Edge Add-ons)、そしてモバイルの App Store・Google Play のみです。検索結果の広告、SNS の DM、QR コード付きの投稿、類似ドメイン(「meta-mask.io」「metamsk.io」など)からのリンクは絶対に使ってはいけません。

偽アプリ・偽拡張の典型的な手口 URL に余計なハイフンや文字が混ざっている、配布元の開発者名が ConsenSys 以外になっている、インストール直後にリカバリーフレーズを要求する、ダウンロード数が不自然に少ない、といった兆候は偽物の強いサインです。2024 年以降、Chrome ウェブストアでは類似拡張が定期的に発見されており、インストール前に開発者名とレビュー欄を確認する習慣が重要です。

ブラウザ拡張機能としてダウンロードする手順

  1. 公式サイトにアクセスする

    ブラウザのアドレスバーに https://metamask.io と直接入力します。検索エンジンの広告枠はクリックしません。広告枠には偽サイトが混入することが過去に複数回報告されています。

  2. 「Download」ボタンを押す

    現在利用中のブラウザが自動判定され、公式ストアのページへ遷移します。判定が異なる場合はページ上部で手動選択できます。

  3. 公式ストアからインストールする

    Chrome ウェブストアでは提供元が「metamask.io」、Firefox Add-ons では「MetaMask」となっていることを必ず確認します。表記が違う場合はインストールしません。

  4. 拡張機能をブラウザに固定する

    インストール後、拡張機能アイコン(パズルピース)からキツネアイコンをピン留めしておくと、誤った類似拡張をクリックする事故を防げます。

スマートフォンにダウンロードする手順

  1. 公式ストア経由でのみ取得する

    iPhone では App Store、Android では Google Play を開き、「MetaMask」で検索します。提供元は必ず「MetaMask」または「ConsenSys Software Inc.」です。

  2. 類似アプリに注意する

    キツネをあしらったアイコンは他にも存在します。開発者名とダウンロード数(通常は数百万以上)、レビュー欄の傾向を確認して区別します。

  3. インストール後は通知権限を最小に

    フィッシング対策として、プッシュ通知経由で届く「取引確認」などを装ったリンクは一切踏まないよう習慣化します。

環境ごとの推奨 DeFi や NFT の操作を PC で行う場合はブラウザ拡張、外出先でのログインや残高確認はモバイルアプリ、という使い分けが一般的です。両方を同じシードフレーズで同期することも可能ですが、フレーズの露出機会が増える点も意識してください。

インストール後の初期設定はどう進めるか?

メタマスクの初期設定は、大まかに「新規ウォレット作成」か「既存ウォレットの復元」の二択から始まります。初めての方は前者、以前フレーズを保存してある方は後者を選びます。どちらの場合でも作業は必ずオフライン状態の静かな環境で、他人に画面を見られないように行ってください。

新規ウォレットを作成する場合

  1. 利用規約に同意する

    初回起動時に、利用規約への同意と匿名の利用状況データ送信の可否を確認されます。データ送信はあとからいつでもオフにできます。

  2. パスワードを設定する

    このパスワードは端末ローカルでのみ使われます。他サービスと使い回さない、12 文字以上の独自のものを推奨します。パスワード自体の総当たり攻撃を防ぐ意味で、大文字・小文字・数字・記号を混ぜます。

  3. シークレットリカバリーフレーズを表示して書き留める

    12 語の英単語が表示されます。紙に手書きし、可能なら耐火・耐水の金属プレートにも転写します。スクリーンショット、クラウド保存、メール送信は避けることが鉄則です。これらはすべて漏えい経路になり得ます。

  4. フレーズの確認テストを完了する

    表示順通りに単語を並べ替えます。これを通過するとウォレットが有効化され、公開アドレスが発行されます。公開アドレスは相手に渡しても問題ありません。

既存ウォレットを復元する場合

過去にメタマスクや互換ウォレット(Rabby、Trust Wallet など BIP39 互換系)で使用していたフレーズがあれば、新規インストール後に「既存のウォレットをインポート」を選び、12 語を順番通り入力するだけで復元できます。同じフレーズからは常に同じアドレスが生成されるため、機種変更時にも有効です。

絶対にしてはいけない四つのこと ① フレーズをオンライン上のメモアプリに保存する、② SNS や DM でフレーズを共有する、③「公式サポート」を名乗る相手にフレーズを入力する、④ 拡張機能の設定画面以外でフレーズを入力する。これらはそのまま資産全額の喪失に直結します。

メタマスクのログイン手順は?

メタマスクのログインは、いわゆる「ID+パスワード」型の中央集権サービスとは仕組みが異なります。各端末にインストールされたウォレットに対して、設定したパスワードでローカル解除するという操作が実質的な「ログイン」になります。外部にログイン状態が記録されるわけではありません。

ブラウザ拡張でのログイン

  1. ブラウザ右上の拡張機能アイコンからキツネのロゴをクリックします。
  2. 表示されたウィンドウにパスワードを入力します。
  3. アカウント残高と接続中ネットワークが表示されればログイン成功です。

dApp へのログイン(ウォレット接続)

  1. 対象 dApp の「Connect Wallet」ボタンを押します。
  2. ウォレット候補から「MetaMask」を選びます。
  3. メタマスクのポップアップで接続要求の内容(アドレス共有のみか、署名まで要求か)を確認します。
  4. 「接続」ボタンで承認し、必要に応じて署名リクエストを処理します。
署名リクエストは必ず文面を読む 接続直後に出てくる署名要求には、資産移動を伴うもの(eth_sendTransaction)、権限付与を伴うもの(approve)、単なる認証用の署名(personal_sign、eip712)など複数の種別があります。内容を読まずに承認すると無制限の権限付与につながることがあります。
ログイン時に出現する主な署名種別
種別何をするか危険度
eth_sendTransaction資産の実際の送金高
eth_signTypedData_v4 (EIP-712)構造化データへの署名(approve 含む)中〜高

モバイルアプリでのログイン

アプリ起動時にパスワードまたは端末の生体認証(Face ID・指紋認証)でロック解除します。ブラウザ内 dApp はアプリ内蔵ブラウザから開くと自動的にウォレットが認識され、外部 Safari・Chrome から開いた場合と比べて接続が安定します。

メタマスクを安全に使ううえで何に注意すべきか?

メタマスクはセルフカストディである以上、資産を守るのは利用者自身です。技術的な機能の多くはメタマスク側で実装されていますが、最終的な判断は常に人間側に委ねられます。日常的に意識したいポイントを以下にまとめます。トラブル対処と合わせて読むと、事故を未然に防ぐ視点が得られます。

ハードウェアウォレットとの併用

中〜長期保有や高額の運用では、メタマスクを Ledger や Trezor などのハードウェアウォレットに接続して使うことを強く推奨します。ハードウェア側に秘密鍵を閉じ込めたまま、メタマスクは署名を依頼するだけのインターフェースとして動作するため、端末感染時の被害を最小化できます。

ネットワーク追加時の注意

独自トークンを扱うために新しいチェーンを追加する場合、RPC エンドポイントは公式ドキュメントに明記された URLしか使わないでください。不審な RPC は取引履歴や残高を改ざんして表示するリスクがあります。

よくあるトラブルと対処法は?

パスワードを忘れた場合

パスワード自体の再発行はできません。ただし 12 語のシークレットリカバリーフレーズが手元にあれば、メタマスクをアンインストール→再インストールし、「既存のウォレットをインポート」から同一ウォレットを復元できます。フレーズと新しいパスワードを設定する流れになります。

残高が 0 と表示される場合

原因の多くは「ネットワークが切り替わっている」「RPC が一時的に応答していない」のいずれかです。画面上部のネットワーク選択で正しいチェーンに切り替えると表示が戻ることがほとんどです。それでも戻らない場合は、RPC を公式のものに手動で切り替えます。

トランザクションが pending のまま進まない

ガス代(ネットワーク手数料)が極端に低いと、混雑時には取り込まれません。メタマスクの「詳細」からガス代を上げる「Speed Up」、もしくは同一 nonce でガス代を上げたキャンセル送信を試します。

別端末で同じアカウントを使いたい

同じシークレットリカバリーフレーズをインポートすれば、どの端末でも同じアドレスにアクセスできます。フレーズは共通の復元キーであり、一度流出すれば端末数に関係なく資産が狙われる点に注意してください。

拡張機能が応答しない・クラッシュする

まずブラウザを再起動し、それでも改善しない場合は拡張機能を一度削除し、公式ストアから再インストールします。シークレットリカバリーフレーズを手元に持っていれば、再インストール後に即座に復元できます。

メタマスクに関するよくある質問

メタマスクのダウンロードは無料ですか?

はい、ウォレットのダウンロードと利用自体は無料です。ただしオンチェーンで取引を行う際にはイーサリアムなど該当ネットワークのガス代が必要になります。

KYC(本人確認)は必要ですか?

ウォレットのインストールと利用に KYC は不要です。ただしメタマスク内で提供されるスワップや日本円建てのオンランプ機能など、一部の外部サービスは、そのサービス提供者の要件によって本人確認を求める場合があります。

メタマスクのログインや画面表示は日本語に対応していますか?

メタマスク本体の UI は日本語を含む多言語に対応しています。拡張機能、モバイルアプリとも設定画面の「言語」から切り替えできます。

ブラウザ拡張とモバイルアプリの両方を使えますか?

可能です。同一のシークレットリカバリーフレーズをインポートすれば、どの端末からでも同じウォレットアドレスにアクセスできます。ただしフレーズの保管責任はそのぶん重くなります。

シークレットリカバリーフレーズを忘れた場合はどうなりますか?

フレーズが完全に失われた状態ではウォレットの復元はできません。これはメタマスク運営を含む第三者も介入不可能な仕様で、セルフカストディの原則そのものです。フレーズは物理的に冗長化して保管することを徹底してください。

メタマスクのログインパスワードとフレーズの違いは何ですか?

パスワードはその端末でウォレットをロック解除するためのもので、端末内部にのみ保存されます。フレーズはウォレットそのものをどの端末からでも復元できる鍵で、全体の所有権に直結する重要情報です。役割がまったく異なります。

メタマスク ダウンロード後、どのネットワークがはじめから使えますか?

インストール直後に使える既定ネットワークはイーサリアムメインネットとテストネット類です。Polygon、Arbitrum、Base などは手動または自動推奨によって追加します。詳細はセキュリティ節のRPCに関する注意を参照してください。

メタマスク ログインに関して公式サポートが個別に連絡してくることはありますか?

いいえ、公式側から DM や電話で連絡が来ることはまずありません。SNS で突然届く「サポート」からのメッセージはほぼすべて詐欺と考えてください。
